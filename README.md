HTTP(S) Proxy server 
===

Installing
-------

    git clone https://github.com/DieOfCode/HTTP-S-Proxy-server.git

    pip install -r requirements.txt
    
Start
----
    python3 main.py --host <host> --port <port>
    
Command
----

    
| Command       | info                | 
| :------------- |:------------------:|
| -p --port     | Specify the port   | 
| -h --host     | Specify the host|
